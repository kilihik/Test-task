# Задание 2: Кулинарное соревнование

## Условие
80 поваров с уникальными уровнями мастерства. Два этапа, на каждом случайные пары. Победитель — выигравший оба этапа. Найти матожидание числа победителей.

## Решение методом индикаторов

Пусть ti — индикатор того, что повар i победил в обоих этапах.
x- победители 
Z - сигма
x = x(t1 + ... + t80) = 80 × x(t1) 

Для конкретного повара с рейтингом R (1 — лучший, 80 — худший):
- В первом этапе побеждает, если соперник слабее: (R-1)/79
- Во втором этапе: снова (R-1)/79
- Вероятность победить в обоих: ((R-1)/79)^2

Усредняем по всем поварам:
x(t1) = (1/80) × Z((R-1)/79)^2 для R = 1..80

Пусть k = R-1, k = 0..79:
x(t1) = (1/80) × Z(k/79)^2 = 1/(80×79^2) × Zk^2 (k=1..79)

Zk^2 (k=1..79) = 79×80×159/6 = 167480

x(t1) = 167480 / (80 × 6241) = 167480 / 499280 ≈ 0.33544

x = 80 × 0.33544 = 26.8352

Ответ: 26,8
