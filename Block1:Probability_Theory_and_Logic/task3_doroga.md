# Задание 3: Одинокая дорога

## Условие
Вероятность появления автомобиля за 30 минут = 0.95. Найти вероятность за 10 минут и за 27 минут. Считаем поток пуассоновским (постоянная интенсивность).

## Решение

Вероятность НЕ увидеть машину за 30 минут: v30 = 1 - 0.95 = 0.05

Для пуассоновского потока: v(t) = e^{-Lt}

Отсюда: e^{-30L} = 0.05

### Для 10 минут:
v10 = e^{-10L} = e^{-30L × (10/30)} = (e^{-30L})^{1/3} = 0.05^{1/3}

0.05^{1/3} = ∛0.05 ≈ 0.3684

p10 = 1 - v10 = 1 - 0.3684 = 0.6316 = 63,2%

### Для 27 минут:
v27 = e^{-27L} = e^{-30L × (27/30)} = 0.05^{0.9}

ln(0.05) ≈ -2.9957
0.9 × (-2.9957) = -2.69613
e^{-2.69613} ≈ 0.0674

p27₇ = 1 - v27 = 1 - 0.0674 = 0.9326 = 93,3%

Ответ: 63,2; 93,3
