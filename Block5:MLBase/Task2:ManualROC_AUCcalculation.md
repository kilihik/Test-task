# Классификатор выдал следующие прогнозируемые метки класса и вероятности принадлежности к классу "1". На основе полученных данных рассчитайте метрику ROC_AUC. Тезисно описать ход решения.


1. Разделил объекты на два класса: положительный (истинная метка 1) и отрицательный (истинная метка 0)
2. Выписал вероятности для всех объектов каждого класса
3. Использовал вероятностную интерпретацию ROC-AUC: вероятность того, что случайный объект класса 1 получит более высокую оценку вероятности, чем случайный объект класса 0
4. Рассмотрел все возможные пары (положительный, отрицательный) — всего 7 × 8 = 56 пар
5. Для каждой пары сравнил вероятности: если P(полож) > P(отриц) — пара считается успешной
6. Подсчитал количество успешных пар — получилось 42
7. Вычислил ROC-AUC = количество успешных пар / общее количество пар = 42 / 56 = 0.75
Ответ: 0.75
